修改 kramed 配置，解决语义冲突

由于 LaTeX 与 Markdown 语法存在冲突（例如在 markdown 中，斜体可以用 * 或者_ 表示，而 LaTeX 也会用到 _ ），所以我们要对 kramed 默认的语法规则进行修改，否则之后会出现很多奇怪的排版样式。
